# Mithas — Handcrafted Coffee Syrups

Website for Mithas Coffee Syrups, a student-run brand at UT Austin crafting Indian dessert-inspired coffee syrups.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

Push to GitHub, then import the repo at [vercel.com/new](https://vercel.com/new).

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Deployed on Vercel
